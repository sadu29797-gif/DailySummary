# 2025年5月学习总结

::: tip
本文记录了2025年5月的学习内容、成果与思考。
:::

## 学习内容

本月主要专注于以下几个方面的学习：

1.  **公共基础课**
    - 高等数学：曲线积分、曲面积分、高斯公式、斯托克斯公式、级数审敛法
    - 大学物理：静电场、热学、晶体管放大电路基础
    - 大学英语：音标跟读、单词积累、听力、阅读、翻译、口语、写作训练
    - 思政类课程：思想道德与法治

2.  **计算机专业课**
    - 前端基础：HTML网页骨架学习与默写
    - 数据库：SQL基础知识学习
    - Java程序设计：实验报告撰写与复习

3.  **实践与其他学习**
    - 家教备课与授课（小学/初中数学）
    - 非遗植物拓染小组作业
    - 习题训练与期中考试复习
